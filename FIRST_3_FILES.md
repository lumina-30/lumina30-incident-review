# First 3 Files
# 最初に読む3ファイル

## Purpose / 目的

This file identifies the first three documents to open when using LUMINA-30 for incident review.

本ファイルは、LUMINA-30をインシデントレビューに使うとき、最初に開く3文書を示す。

## 1. LUMINA-30 Boundary Check / LUMINA-30 境界チェック

File:

[boundary-check/LUMINA-30_Boundary_Check.md](./boundary-check/LUMINA-30_Boundary_Check.md)

Use this to answer the central boundary question:

中核となる境界問いに答えるために使用する。

> Was human final refusal effective before irreversible impact?

> 不可逆的影響の前に、人間の最終拒否権は実効的だったか。

## Minimal Rule / 最小ルール

If effective human refusal cannot be verified, do not claim that effective human refusal was demonstrated.

実効的な人間の拒否を検証できない場合、「実効的な人間の拒否が示された」と主張しない。
