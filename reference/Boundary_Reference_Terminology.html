Boundary Reference Terminology

LUMINA-30 incident review public reference material.

Purpose

This file fixes the preferred terms for describing LUMINA-30 without overclaiming its status.

Preferred Terms

Use these terms:

  • non-binding

  • descriptive

  • boundary reference

  • post-incident review

  • effective human refusal

  • before irreversibility

  • refusal pathway

  • operationally effective

  • verifiable records

  • procedural invalidity for boundary-review purposes

Central Phrase

effective human refusal before irreversibility

Core Question

Was human refusal authority effective before irreversible impact?

Use With Care

The following terms may be used only with qualification:

  • criterion

  • framework

  • boundary condition

  • procedural invalidity

  • review classification

Avoid

Avoid describing LUMINA-30 as:

  • binding law;

  • mandatory standard;

  • certification scheme;

  • enforcement mechanism;

  • anti-AI ideology;

  • final legal test;

  • regulatory replacement.

LUMINA-30

Recommended Sentence

LUMINA-30 is a non-binding, descriptive boundary reference for reviewing whether effective human refusal remained available before irreversible AI impact.
